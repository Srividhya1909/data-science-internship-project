# Task 3: Exploratory Data Analysis (EDA) on Titanic Dataset

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on the Titanic dataset. The objective is to uncover patterns, trends, and relationships within the data and identify the factors that influenced passenger survival.

## Dataset Description

The dataset contains information about Titanic passengers with the following features:

- Survived
- Pclass
- Name
- Sex
- Age
- SibSp
- Parch
- Ticket
- Fare
- Cabin
- Embarked

## Objectives

- Understand the structure of the dataset
- Analyze passenger demographics
- Identify factors affecting survival
- Discover patterns using statistical analysis and visualizations
- Present findings through data-driven insights

## Data Analysis Performed

### Dataset Overview
- Examined dataset dimensions
- Checked data types
- Displayed sample records

### Missing Value Analysis
- Identified missing values in Age, Cabin, and Embarked columns
- Applied appropriate preprocessing techniques

### Statistical Summary
- Calculated mean, median, minimum, maximum, and standard deviation
- Analyzed numerical features such as Age and Fare

### Exploratory Visualizations

The following visualizations were created:

1. Survival Distribution
2. Passenger Class Distribution
3. Gender Distribution
4. Age Distribution
5. Fare Distribution
6. Survival by Gender
7. Survival by Passenger Class
8. Survival by Age Group
9. Family Size Distribution
10. Survival by Embarkation Port
11. Average Fare by Passenger Class
12. Correlation Matrix

## Key Findings

- Female passengers had a significantly higher survival rate than male passengers.
- First-class passengers were more likely to survive than passengers in lower classes.
- Most passengers were young adults between 18 and 35 years old.
- Higher ticket fares were generally associated with better survival rates.
- Passenger class showed a strong relationship with survival outcomes.
- Family size and embarkation location also influenced survival probability.
